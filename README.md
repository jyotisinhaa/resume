# Resume

My professional resume hosted as a web page.

## Overview

This project displays my resume PDF in a clean, full-screen web interface.

## Files

- `index.html` - Main HTML page that displays the resume
- `resume.pdf` - Resume in PDF format

## How to View

### Local Testing

1. Start a local server:
   ```bash
   python -m http.server 8000
   ```
2. Open http://localhost:8000/index.html in your browser

### Alternative

Simply open `index.html` directly in your web browser.

## Deployment

This can be deployed to any static hosting service like GitHub Pages, Netlify, or Vercel.
